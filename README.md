# MyOpenBot

基于 Next.js + TypeScript + Tailwind CSS 的 Todo List 应用。

## 功能特点

- ✅ 添加、完成、删除任务
- ✅ 任务筛选（全部/进行中/已完成）
- ✅ 本地存储持久化
- ✅ 响应式设计，支持手机访问
- ✅ 现代简洁的 UI 设计

## 技术栈

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
